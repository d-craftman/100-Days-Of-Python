# Day 25: Aggregate Functions & Financial Analytics

## 🎯 Objectives
- Perform data aggregation on nested JSON dictionaries.
- Use `max()`, `min()`, and `sum()` for business reporting.
- Calculate Average Project Value (APV) for the studio.

## 🛠️ The Build: Craftroom Analytics Dashboard
You can't manage what you don't measure. Today, we move from "Logging" to "Analyzing." We’ve built a **Financial Module** that scans the entire JSON database to provide the studio with its three most important metrics: Total Pipeline Value, Average Project Size, and our Highest-Paying Client.

### Key Features:
- **Revenue Aggregation:** Real-time calculation of total studio earnings.
- **Outlier Detection:** Identifying the most and least expensive projects in the archive.
- **Statistical Insights:** Providing the average budget to help with future pricing strategy.

## 🚀 Technical Takeaway
Aggregate functions allow us to see the "Big Picture." Turning thousands of rows of data into three meaningful numbers is the core of Backend Engineering.
